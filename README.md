# FinPilot AI Dashboard

A modern, AI-powered personal finance intelligence dashboard built with HTML, CSS, and JavaScript.

## Features

- **Financial Health Score** - Real-time assessment of your financial position
- **Net Worth Tracking** - Monitor wealth growth over time
- **Investment Portfolio** - Track investments across multiple asset classes
- **AI Fraud Detection** - Smart alerts for suspicious transactions
- **Expense Forecasting** - Predict upcoming expenses by category
- **Goal Center** - Track financial goals with visual progress
- **Credit Score Analysis** - Monitor and improve credit health
- **Tax Intelligence** - Maximize tax savings with personalized recommendations
- **AI Copilot** - Interactive AI assistant for financial queries
- **Smart Notifications** - Real-time alerts for important financial events

## Design

- Dark mode UI with modern gradient accents
- Responsive dashboard layout
- Real-time data visualization with Chart.js
- Color-coded alerts and metrics
- Accessible design with semantic HTML

## Deployment on Vercel

### Option 1: Using Vercel CLI

```bash
npm install -g vercel
vercel
```

### Option 2: Connect GitHub Repository

1. Go to [Vercel Dashboard](https://vercel.com/dashboard)
2. Click "New Project"
3. Import this GitHub repository
4. Vercel will auto-detect it as a static HTML project
5. Click "Deploy"

### Option 3: Manual Deployment

1. Visit [vercel.com](https://vercel.com)
2. Sign up or log in
3. Drag and drop the project folder
4. Vercel will automatically deploy

## Live Preview

After deployment, your dashboard will be available at: `https://your-project-name.vercel.app`
